Description

HRMRIF : Human Resource Management System RIF.
Creation of a human resources management system. The application is devised in two parts, a backend part with Spring boot + micro services and a frontend part with Angular.
The frontend side was developed with Angular and the backend is composed of Spring Boot applications organized according to the architecture of the micro service.
List of micro services :

  1. ConfigurationService : it is a micro service which contains a configuration file, it is linked to a configuration repository on git.
  2. ServiceDiscovery : it is a micro service which contains all the information of all the other micro services up / down ...
  3. ServiceGateway with Zull : It acts as a proxy; this is the first door to use the app. There is also the management of authentication.
  4. UserManagement : This micro service is intended to manage the users (HR Director, Service Agent, Employee) of the RHManagement-RIF application.

Technologies used :

Java, Spring Boot, Angular, PostgreSQL, microservices, API Nexmo.
