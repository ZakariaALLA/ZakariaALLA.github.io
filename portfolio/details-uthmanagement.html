Project information

  • Project date: Jul. 2019, Aug. 2019

Description

Desktop Application To Manage the UTH Group.
UTH Management is a small system that will allow the management of the UTH GROUPE company, for example to add customers, modify their information and delete them, Also the organization of the stock, the user can add, delete, modify or search for a product in the database, as he can see each product with its details.
Thus the user can create an invoice, an estimate, delivery note and purchase order for each customer in Excel form as he can modify on this Excel file from the application, there is also a part for suppliers and the accounting.
It is therefore an application that ensures the organization of the UTH Groupe company.

Technologies used :

Java, Java FX, SQLite, Apache POI.
